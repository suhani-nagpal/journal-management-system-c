# Journal Management System in C

A simple command-line based journal application built in **C** using file handling.  
This project allows users to write, view, search, and delete journal entries stored in a file.

---

## Features
- Add journal entries (date, title, content)
- View all saved entries
- Search entries by keyword or date
- Delete specific entries
- Persistent storage using files

---

## Concepts Used
- File handling (`fopen`, `fclose`, `fgetc`, `fgets`, `fprintf`)
- Functions
- Loops and conditional statements
- Strings
- Git & GitHub version control
